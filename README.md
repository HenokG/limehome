# How to use

Copy `.env.example` to `.env` and fill in the values

Run `docker-compose up` and you should be good to go

# Stack

    * Typescript
    * Express
    * Mikro-ORM
    * Sqlite
    * Pino
    * Docker
    * Docker Compose
    * Jest
    * Supertest
    * ESLint
    * Prettier
    * Husky
    * Commit Lint
    * Github Actions
